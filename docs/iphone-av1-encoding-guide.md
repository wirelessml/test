# iPhone で AV1 動画エンコード ガイド

## 概要

iPhoneでAV1コーデックの動画を作成する方法。
iPhone単体ではAV1エンコードのネイティブ対応がないため、クラウド変換サービスを利用する。

## 前提知識

- iPhone はAV1の**再生（デコード）**はiPhone 15 Pro以降で対応
- AV1の**エンコード**はiOSネイティブでは非対応
- ハードウェアエンコーダも未搭載

## 方法: CloudConvert を使う

### 手順

1. Safari で https://cloudconvert.com にアクセス（アカウント作成が必要）
2. 「Select File」で動画ファイルを選択
3. 出力形式を **MP4** に設定
4. スパナ（設定）アイコンをタップ → **Video Codec → AV1** を選択
5. 「Convert」をタップ
6. 完了後「Download」で保存

### 無料枠

- 1日25分（動画の合計時間）まで無料
- それ以上はクレジット購入が必要

## AV1エンコードの確認方法

### iPhoneのファイルアプリで確認（推奨）

1. 「ファイル」アプリを開く
2. ダウンロードした動画ファイルを**長押し**
3. 「情報を見る」をタップ
4. 「コーデック」欄に **libaom-av1** と表示されていれば成功

### VLC for iOSでの確認

- VLCの再生画面ではコーデック情報の直接確認が難しい
- ファイルアプリでの確認を推奨

## 実際の変換結果（2026年3月29日）

| 項目 | 値 |
|------|-----|
| 元形式 | MOV |
| 変換後 | MP4（AV1） |
| コーデック | libaom-av1 |
| 解像度 | 360 × 480 |
| 再生時間 | 00:10 |
| ファイルサイズ | 360 KB |
| エンコーダ | Lavf60.16.100（ffmpeg系） |
