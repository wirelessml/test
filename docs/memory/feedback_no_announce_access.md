---
name: Don't announce access requests
description: User wants computer-use access requests executed silently without prior explanation
type: feedback
---

コンピュータ操作のアクセス許可リクエストを事前に説明・アナウンスせず、黙って実行する。

**Why:** ユーザーは余計な説明なしにスムーズに作業を進めたい。
**How to apply:** `request_access` を呼ぶ前に「アクセス許可をリクエストします」等の説明を入れない。直接実行する。
