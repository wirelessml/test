# NVMe 世代交代で 1.5 倍速 Note

> 2026/04/29 Substack Note 投稿用
> 文字数: 約 480 字

---

7 年前の Plextor が死んで、後継の Acer FA100 512GB に替えた。Plextor 時代と速度はそんなに変わらない、と書きそうになって、CrystalDiskMark を見直したら **1.5 倍速くなっていた**。

| | Plextor PX-256M8PeGN (2018) | Acer FA100 (2025) |
|---|---|---|
| 公称 SEQ Read | 2,300 MB/s | 3,300 MB/s |
| 実測 SEQ Read | (生前記録なし) | **3,374 MB/s** |

両方とも **PCIe Gen3 x4 NVMe**。同じレーン規格のまま、5 年で NAND とコントローラが進化して、Gen3 x4 の理論天井 (3,940 MB/s) に **85% efficiency** で到達した。

**ここで重要なのは、しゅん先生 PC のマザボが Intel Z370 (2017 年 Q4) ということ**。Z370 は Gen3 までしか対応していない。PS5 用の Gen4 SSD (7,000 MB/s クラス) を買っても、このスロットでは **3,500 MB/s で頭打ち**になる。

つまり 7 年前の i7-8700K + Z370 のマザボでも、SSD の世代交代だけで Gen3 の天井まで使い切れる。マザボ買い替え (= 数万円) せずに体感速度が上がるのは大きい。

実質 ¥10,267 (はばタンPay+ 50% プレミアム) で 容量 2 倍 + 速度 1.5 倍。NAND 高騰 2026 年でこれは滑り込みセーフだった。

## 事実確認メモ

- Plextor PX-256M8PeGN: 2018 年製、公称 SEQ R 2,300 MB/s、Gen3 x4 NVMe (事実、現物ラベル確認 2026-04-24)
- Acer FA100 512GB: NVMe 1.4、公称 SEQ R 3,300 MB/s、Gen3 x4 NVMe (事実)
- CrystalDiskMark 9.0.2 計測 (2026-04-29 16:35): SEQ1M Q8T1 R/W = 3,374.22 / 2,826.92 MB/s (実測)
- Gen3 x4 理論値: 3,940 MB/s (規格)
- マザボ: BIOSTAR RACING Z370GT3 想定、Intel Z370 チップセット (2017 年 10 月リリース、CFL 対応の最初のチップセット)
- Z370 の M.2 サポート: Gen3 x4 まで (Gen4 は Intel 11th Gen / Z490 以降、AMD なら X570 以降)
- 価格: 大西ジム新長田で店頭 ¥15,400、はばタンPay+ 第 5 弾 50% プレミアムで実質 ¥10,267 (2026-04-27)
