## 学んだこと / 気づき

### 1. ping ≠ オフライン
Windows Firewall は ICMP デフォルトブロック。 `ping` 不通でも `ssh` は通る可能性。 LAN 検証は SSH まで実行すべき。

### 2. Microsoft の「30 年前リバイバル」 2 連発
edit コマンド（1981→2026 OS バンドル復活）と カイル（1997→2026 AI 化復活）が **同じ Windows 11 25H2 上で同居**している。Microsoft のノスタルジア戦略が AI 時代に明確化。

### 3. Word docx は string replacement で十分編集可能
複雑な XML 構造でも `document.xml` の直接編集（位置ベース + 文字列置換のハイブリッド）で 4 セクション同時修正可能。 paragraph 単位の置換 + 表全体削除も実用範囲。

### 4. はばタンPay+ の「事業所単位」加盟
同一法人・同一住所でも、事業所（店頭販売 / 飲食店）が分かれていれば加盟有無も別。 1 度断られても入口を変えて再挑戦の価値あり。

### 5. SSH の Session 0 isolation
Windows OpenSSH のセッションは Session 0（Services）。 GUI アプリ起動はユーザー Session 6（Console）で表示されるため、 SSH 経由ではデスクトップキャラを起こせない。 物理ログオン or Remote Desktop 経由が必要。
