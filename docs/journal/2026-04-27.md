## 重要な学び（ナレッジ保存）

### 1. Tailscale Funnel は最速の Next.js 即時公開手段

- **コマンド 1 つで HTTPS 公開**: `tailscale funnel --bg 3000`
- 制約:
  - Mac 起動中のみ稼働（スリープで停止）
  - 初回有効化はブラウザ承認必要（https://login.tailscale.com/f/funnel?node=...）
  - dev server (`next dev`) 直のため本番性能保証なし
- 利用シーン: **Substack ネタの短期 PR**、**24 時間限定の note 記事連動**、**iPhone から外部接続テスト**

### 2. dev server に API キーを渡さなければ Tailscale Funnel 公開でも安全

- `new Anthropic()` は環境変数 `ANTHROPIC_API_KEY` を読む実装
- dev server プロセスに渡さなければ Anthropic SDK は認証エラー（HTTP 500）
- → **API ルートが「セキュリティホール」ではなく「死んでいる機能」**として公開可能
- 自分が使う時だけ別シェルで `ANTHROPIC_API_KEY=sk-ant-... npx next dev` 起動

### 3. HyperFrames は仲さんの動画パイプラインに完璧フィット

- HTML + CSS + GSAP timeline → MP4 の宣言的記述
- 1080×1080 / 15 秒 / 474KB の超軽量出力
- React/Babel 不要、Pure HTML
- 既存 PIL + ffmpeg 手作業より遥かに洗練（テンプレートファースト）
- フォント警告: `Hiragino Sans GB` 等が deterministic font に未マッピング → Chrome 既定にフォールバック（実用上問題なし）

### 4. ひらがな → ローマ字変換は拗音 / 促音 / 撥音の 3 段階

- 拗音（きゃ→kya）: 2 文字パターンマッチを最優先
- 促音（っ→次の子音重ね）: 先読みして次トークンの先頭子音を取得
- 撥音（ん→n / nn）: 文脈依存、母音/y/ん の前は nn、それ以外は n
- 長音 ー → `-` でローマ字に直接出る
- これらを正しく実装すれば、ひらがな上級レベルの全文をタイピング教材化可能

### 5. NAND 高騰の体感は 128GB 中古 ¥4,400 で確定

- 2 年前 ¥800-1,500 だった 128GB 中古極上が ¥4,400
- 1TB NVMe ¥20,000+ と整合する 3-5 倍値上がり
- Substack 記事の冒頭エピソード化に最適な実例

### 6. バイラル AI 物語の構造定石

「Vision Pro 寝転びコーディング」型の共通点:
- 具体数字（時間、金額、件数）
- 父子 / 業界外人 / 弱者の主人公
- 「映画を見てる」のオチ式ミスリード
- ROI が極端（例: 9 日でハードウェア回収）

→ 仲さんが書く時は **数字を実体験ベース**にすればこの定石を踏襲できる（SSD 記事も「6 時間 41 分」「BIOS 100 設定中の 1 行」が既にこの形）。
