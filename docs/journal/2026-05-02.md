## 学んだこと

- **`brew install --cask <gui-app>` は Claude bash で完走できないことがある** — cask の postinstall が `installer pkg` を sudo 経由で叩く必要があり、Claude のセッションには tty がない（`sudo: a terminal is required to read the password`）。ユーザー側 ! プレフィクスで実行が定石
- **Wireshark cask は ChmodBPF helper を同梱**: Wireshark 単体導入だけで `/dev/bpf*` 永続化 + `access_bpf` グループ整備までやってくれる（`wireshark-chmodbpf` cask 単独より楽、Wireshark 本体も付いてくる）
- **`do shell script with administrator privileges` の罠**: 5/1 夜に試した方式は osascript 自体は exit 0 を返すが、内部で `nohup` が `Inappropriate ioctl for device` で落ちて GUI アプリは実起動しない。GUI アプリを sudo 経由で起動するには別アプローチ（`open -a App.app` + helper 等）が必要だった
- **Tailscale 4 トンネル分の utun デバイスが見える**: Tailscale Funnel デバッグ時は `tshark -i utun0` で WireGuard パケットの観察が可能
