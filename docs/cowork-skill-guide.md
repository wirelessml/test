# Cowork用 SKILL.md 活用ガイド

## 概要

SKILL.mdを1枚入れるだけで、Coworkが毎回同じ品質で動く。
一度作れば、毎回の説明がゼロになり、呼びかけだけで同じ品質が毎回出る。

## SKILL.mdとは

| ファイル | 役割 |
|---------|------|
| CLAUDE.md | プロジェクト全体のコンテキスト |
| SKILL.md | 特定タスクの手順書・スキル定義 |

## 配置場所

```
.claude/skills/my-workflow.md
```

## 基本テンプレート

```markdown
---
name: my-workflow
description: "（このスキルが何をするか1行で書く。例: X投稿の下書きを3パターン生成する）"
---

# 役割
あなたは（職種・役割）の専門家です。
（ターゲット）に向けて（目的）を達成するために動きます。

## 禁止事項
- （やってはいけないこと1）
- （やってはいけないこと2）
- 抽象的な表現は使わない（具体的な固有名詞・数字を必ず入れる）

## 出力フォーマット
（出力の形式・文字数・構成を書く）

## 参照ファイル
- knowledge/（参考にするファイル名）
- memory/（記憶させたいファイル名）
```

## 使い方

1. 上のテンプレートをコピーして自分の情報に書き換える
2. `.claude/skills/` フォルダに保存
3. Coworkから呼びかけるだけで、定義した通りに動く

## メリット

- 毎回の指示出しが不要になる
- 出力品質が安定する
- チームで共有すれば誰でも同じ品質で作業できる
