Claude Computer Use（macOS）セットアップ & 利用ガイド

2025年3月23日発表。Claude がユーザーの許可のもと Mac を直接操作し、デスクワークを代行する機能。

⚠ 現在のステータス：リサーチプレビュー - 対象プラン：Pro / Max のみ - 対応OS：macOS のみ（Windows / Linux は未対応） - Claude デスクトップアプリの起動が必須 - 正式リリースではないため、機能や仕様が変更される可能性がある

1. 前提条件

項目 要件
ステータス リサーチプレビュー（正式版ではない）
プラン Claude Pro または Max
OS macOS のみ（Windows / Linux は未対応）
アプリ Claude デスクトップアプリ（最新版）の起動が必須
ブラウザ操作 Google Chrome（拡張機能が必要）

2. 機能の概要

Claude の「Cowork」タブ内にある Dispatch（ディスパッチ） 機能を使うと、Claude が以下を実行できる：

  • アプリの起動・切り替え
  • ブラウザでの検索・ナビゲーション・フォーム入力
  • スプレッドシートへの値入力
  • ファイル操作（読み取り・書き込み）
  • その他一般的なデスクトップ操作

3. セットアップ手順

3.1 Claude デスクトップアプリを開く

上部タブで 「Cowork」 を選択し、左サイドバーから 「Dispatch」 をクリック。

「ディスパッチの準備」画面が表示される。

3.2 各設定項目を有効化する

以下の順番で設定を進める：

① Claudeにファイルへのアクセスを許可する

  • トグルを オン にする
  • 「完了」をクリックすると macOS がアクセシビリティの許可を求める
  • システム設定 > プライバシーとセキュリティ > ファイルとフォルダ で Claude を許可

② このコンピュータをスリープさせない

  • Dispatch の実行中にスリープを防止する設定
  • 長時間タスク（リサーチ、データ入力など）を任せる場合は オン を推奨

③ Claude in Chrome をインストール

  • 「インストール」ボタンをクリック
  • Chrome 拡張機能がインストールされる
  • これにより Claude がブラウザ内でナビゲート、クリック、フォーム入力を行えるようになる
  • 注意: Chrome が既定ブラウザとして設定されている必要がある

④ Claudeにコンピュータの制御を許可

  • トグルを オン にする
  • macOS のアクセシビリティ許可が求められる
  • システム設定 > プライバシーとセキュリティ > アクセシビリティ で Claude を許可
  • これにより、クリック・キーボード入力・アプリの起動が可能になる

⑤ コネクタの確認

  • 「すべてのコネクタがオンです」と表示されていれば OK
  • 認証済みの外部サービス（Google、Slack など）との連携状態を確認

3.3 設定完了の確認

すべての項目が有効になったら、左サイドバーの 「+ 新しいタスク」 からタスクを作成できる。

4. 基本的な使い方

4.1 タスクの作成

  1. 「+ 新しいタスク」 をクリック
  2. 自然言語でやりたいことを入力する（例：「Googleスプレッドシートを開いてA1にタイトルを入力して」）
  3. Claude がタスクを分解し、ステップごとに実行する

4.2 タスクの例

カテゴリ 具体例
ブラウザ操作 「〇〇のサイトを開いて情報をまとめて」
データ入力 「このCSVの内容をGoogle Sheetsに転記して」
リサーチ 「競合3社の料金プランを調べて比較表を作って」
ファイル管理 「デスクトップのPDFを読んで要約して」
アプリ操作 「Numbersを開いて新しいシートを作成して」

4.3 タスクのスケジュール

  • 左サイドバーの 「予定済み」 からタスクの予約実行が可能
  • 定期的なタスク（日次レポート作成など）に活用できる

5. macOS の権限設定まとめ

Computer Use を利用するには、macOS のシステム設定で以下の権限を付与する必要がある：

システム設定 > プライバシーとセキュリティ
権限カテゴリ 用途
アクセシビリティ クリック・キーボード入力・アプリ制御
ファイルとフォルダ ファイルの読み書き
画面収録（必要に応じて） 画面の状態を認識するため

6. セキュリティ上の注意点

  • Claude は ユーザーの許可なく 操作を行わない。各操作前に確認が入る
  • パスワード入力や決済操作は慎重に扱うこと
  • 機密情報が含まれるアプリ・サイトを操作させる場合はタスク内容を限定する
  • 「コンピュータの制御を許可」は使わないときは オフ にしておくことを推奨
  • Chrome 拡張経由の操作はブラウザのシークレットモードでは動作しない

7. トラブルシューティング

症状 対処法
タスクが実行されない macOS のアクセシビリティ権限を再確認
ブラウザ操作ができない Chrome 拡張「Claude in Chrome」がインストール済みか確認
途中でスリープする 「このコンピュータをスリープさせない」をオンにする
画面認識がおかしい ディスプレイ解像度の変更やマルチモニター構成を見直す
コネクタが認証切れ Dispatch 設定画面でコネクタを再認証

8. Cowork タブの構成

Claude デスクトップアプリの上部タブは3つ：

タブ 機能
チャット 従来のテキストチャット
Cowork Computer Use（Dispatch）によるデスクトップ操作
コード Claude Code（コーディング支援）
