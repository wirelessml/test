# しぶのAI活用の全体像

## ツール構成
```
iPhoneから:
  Dispatch → MacBook Air (Claude Cowork) → CupCut(動画編集)

Discord:
  URL投稿 → しぶ次郎Bot → OpenClaw → Obsidian

MacBook Air:
  Claude Code CLI → 開発・自動化
  OpenClaw → 情報整理・要約
  Obsidian → ナレッジ保存

LINE:
  AIミニマリストしぶ Bot → ファン向けAI
```

## 各ツールの役割
- Claude Code: 開発エンジン、コーディング、自動化
- Claude Cowork + Dispatch: リモートPC操作、動画編集
- OpenClaw: URL要約、情報自動整理、Obsidian連携
- しぶ次郎(Discord Bot): AI秘書、調べもの、メモ整理
- AIミニマリストしぶ(LINE): ファン向けのAIチャットボット
- Obsidian: 全情報の永続保存・整理

## ハードウェア
- MacBook Air M5 x2（424,800円x2）
  - 1台: 持ち歩き用
  - 2台: 自宅Claude常時稼働用
- Apple Vision Pro: 仮想大画面
