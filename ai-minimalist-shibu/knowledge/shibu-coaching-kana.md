## コーチングの流れ

### 1日目（8時間）

#### オンライン面談（事前）
- LINEで写真・申込動機を事前共有
- しぶの第一印象：「休めですね。この辺りとかすごい綺麗」

#### 片付け前ルームツアー
- 玄関 → 靴箱 → 廊下 → 洗面所 → トイレ → リビング → キッチン
- 全収納を開けて確認、問題点を可視化

#### 自己紹介・目標設定
- かなさんの自己紹介と応募動機を改めて確認
- 目標：「物理的にも精神的にもシンプルに、思考クリアに残りの人生を気持ちよく生きたい」

#### 服の片付け（メインイベント）
1. **全出し**: クローゼット・タンス・押入れから全ての服を出す → ベッドが埋まるほどの量（推定160〜200着）
2. **1巡目**: 捨てるものをゴミ袋へ → 4袋
3. **2巡目**: いると言った服をもう一度見直し → さらに減らす
4. **3巡目**: 吊した服を数えて現実を見る → 約78着
5. **押入れ・タンスの服も統合**: 冬物も含めてオールシーズン全80着に絞る
6. **タンス撤去**: 大型タンス1個を撤去、収納を1箇所に集約

#### 鞄・傘・ベルト・スリッパ
- 押入れの鞄類を全出し → 不要品大量
- 傘: 5本 → 2本（日傘兼用1本 + 雨傘1本）
- ベルト: 半分に削減（ベルトレスの流れも提案）
- 来客用スリッパ: 全部処分

#### 押入れの片付け
- 防災アイテムの整理: 複数箇所に散在していたものを1箇所に集約
- 防災備蓄ルール: **3-5-7日**（最低3日、推奨5日、7日以上は過剰）
- ロウソクは防災に不向き（地震で火災リスク）→ 手回し式ライト推奨
- 思い出系は最後に（時間がかかるため）
- ひな人形は子供に確認してもらう

#### 本の片付け
- 積読本を中心に約半分に削減
- 紙の本は最悪手放しても買い直し可能（古本なら数百円）
- 思い出のアルバムは一旦保留

#### 靴箱の片付け
- かなさんの靴: 18足 → 12足（6足削減）
- 靴箱の敷き物も撤去（ストレスの元、靴箱は元々汚れるもの）
- 2段収納グッズも撤去
- 家族の靴も一部事前確認で処分

#### 1日目の成果
- **ゴミ袋大量** + 大型家具（タンス）撤去
- 「こんなに減らせるなんて思っていなかった」

- 大型家具（タンス等）を0円で出品 → 取りに来てもらう方式
- 利用料無料、粗大ゴミ代（約3,000円〜）が浮く

## しぶの具体的なアドバイス・手法

### 片付けの順番
1. **服** → 翌朝から効果を実感できる（服選びが楽になる）
2. **紙の本** → 服を減らした流れで
3. **靴箱** → 服のコーディネートと合わせて判断
4. **収納の中身**（ケーブル、防災、書類等）
5. **キッチン**（食器、調理器具、保存容器）
6. **思い出のもの** → 最後（時間がかかるため）

### 基本メソッド
- **全出し**: 収納の中身を全て出して向き合う。出すだけで「こんなにあったの？」と気づく
- **残すものを選ぶ**（捨てるものを選ぶのではなく）
- **複数巡回**: 1巡目は甘くなりがち。2巡、3巡と繰り返すことで基準が厳しくなる
- **短期集中**: 10年かけてちょこちょこやるより、数時間で一気にやる方が効果的
- **収納ごと手放す**: 中身を減らすだけでなく、タンスや棚自体をなくす
- **数値目標**: 服は80着以下（理想は60着）、靴は人数分+α

### 服の減らし方
- **全部1箇所に集める**（散在していると全体量が把握できない）
- **オールシーズン吊るせる量に**: タンスを使わない
- **「着れる」ではなく「着たい」**: 着れるから残す→永遠に減らない。着たいものだけ残す
- **ワンマイルウェア不要**: 近所用・部屋着用を分けない。お気に入りを普段から着る
- **色違い買いに注意**: 結局着用頻度に差が出る
- **服のカバーは全捨て**: 通気性が悪くなる、存在を忘れる
- **セール品は買わない**: 安いから買ったものは着なくなる（定価で買うかどうかが基準）
- **1軍だけ残す**: 旅行に持っていくか、災害時に持ち逃げするか
- **上限を決める**: 80着と決めたら買ったら1つ手放す

### 防災の考え方
- **3-5-7ルール**: 最低3日分、推奨5日、7日以上は過剰
- ロウソクは危険（地震で火災リスク）→ 手回し+ソーラー式ライト1個で十分
- 防災アイテムは**1箇所に集約**（枕元近く推奨）、散在させない
- ローリングストック法の推奨（普段食べるものを備蓄兼用に）

### ふるさと納税の注意
- 「節税」ではなく「節約」（払う税金額は同じ）
- 物の返礼品は溜まりがち → **経験に使う方がおすすめ**
- 洗剤等の大量ストックは保管コスト（家賃の一部）を食っている

### その他のアドバイス
- **収納は半透明がおすすめ**: 中身が見えないと溜め込む、透明だと生活感が出る
- **身長より高い家具は置かない**: 地震リスク＋圧迫感
- **ケーブルの予備は基本不要**: 100均・コンビニで即入手可能
- **古いスマホは3大キャリアに持ち込めば無料回収**
- **ハイブランドの箱は捨てる**: メルカリで50〜100円程度（場所代の方が高い）
- **プリンター**: 月2回以下の使用なら家に置かない（セブンネットプリント推奨）
- **売る・譲るより捨てる**: 売る前提だとリバウンドする（どうせ売れるからまた買う）
- **1万円以下のものは売らずに捨てた方が時給的に効率的**
- **食器は家族人数+2を基準に**
- **ジモティ**: 大型家具の無料処分に最適（利用料無料、当日〜翌日引取りも可能）
