# しぶの住居遍歴

## 実家（〜19歳）
- 邸宅に住んでいた（父が投資家で裕福）
- 毎週新作ゲームを買う「マキシマリスト」家庭
- 父の自己破産で一変 → 邸宅は「汚部屋」に

## 最初の一人暮らし（19歳〜）
- 実家から徒歩5分、風呂なしワンルーム
- 家賃19,000円
- ジムで入浴
- 「安い家賃の狭い部屋で生活するためにはモノを減らすしかない」

## 福岡ワンルーム時代（〜2025年）
- 福岡市のワンルーム約30㎡
- 家賃2万円（7年間）
- 白い壁、白い床、最小限の家具
- カーテンなし、カーペットなし、時計なし、テレビなし、冷蔵庫なし
- 収納が一切ないワンルーム（モノの量が丸わかり）

## テスラ車上生活（2026年〜）
- テスラ モデルYを新居として家を手放す
- 「車を買ったというより生活空間を買った」
- 月の生活費5.8万円（家賃ゼロが最大の理由）
- 住民票のために家は残しているが1ヶ月帰っていない

## 住居の進化
- 邸宅 → 風呂なしワンルーム → 30㎡ワンルーム → テスラ車内
- 面積: 数百㎡ → 30㎡ → 車内
- 「住む場所が小さくなるほど、自由が大きくなった」
- ミニマリズムの到達点: 家すら持たない暮らし
