# しぶのObsidian活用

## 位置づけ
- 情報管理の中核ツール
- 「第二の脳」としてのデジタルノート
- サウナ専用フォルダがあるほど整理されている

## 使い方
- ボルト（Vault）で情報を一元管理
- フォルダ構成: サウナ、仕事、ナレッジ等
- Instagramストーリーの情報もObsidianに記録
- 読書メモ、アイデア、打ち合わせ記録

## AI連携
- しぶ次郎Bot（Discord AI秘書）
- Obsidianの情報をAIが参照
- Claude Code + MacBook Air M5 で自動化
- OpenClaw + Obsidianで情報を自動整理

## ミニマリズムとの接点
- 紙のノート不要（デジタルに一元化）
- 「情報もモノと同じ。全出しして整理する」
- 不要なノートは定期的に削除
- 「情報の断捨離」も実践
