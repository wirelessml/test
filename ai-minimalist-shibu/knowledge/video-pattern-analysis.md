# しぶチャンネル動画パターン分析

> 21本のトランスクリプトから抽出した構造・フレーズ・クライアントの共通パターン

## 1. 動画の構成テンプレート

```
冒頭（0〜2分）
├─ 「お邪魔します」（全動画共通）
├─ 衝撃表現（「やばい」「大変なことになってます」「悪の光景」「事件発生」）
├─ 部屋の状態を即座に視覚的に描写
└─ クライアントの自己紹介・応募動機の読み上げ

中盤（片付け本編）
├─ 全出し（コアメソッド）
├─ 指導フレーズの繰り返し
├─ クライアントとの対話・気づきの瞬間
├─ ゴミ袋の山・不用品の可視化
└─ 日ごとの区切り（「2日目です」「おはようございます」）

終盤
├─ ビフォーアフターの公開（「じゃん」「なんということでしょう」）
├─ クライアントへの励まし
├─ 「ご視聴いただきありがとうございました」
└─ LINE登録のお知らせ（完全固定テンプレート）
```

### エンディングの固定テンプレート（全動画共通）
- LINE追加で無料特典10個:
  - YouTube未公開の片付け動画・ルームツアー動画5本
  - アプリアイコン画像150個＋壁紙セット
  - 物を手放すか迷った時の30の質問リスト
  - しぶと1対1でZOOMできる相談会
  - ミニマリスト部屋のリンク集
- 「配布は予告なく終了する可能性があるので受け取りを早めに」

→ **全動画がLINE登録への導線として機能**

## 3. クライアントの共通属性

### 人口統計

| 属性 | パターン |
|------|---------|
| 性別 | **女性16名、男性2名**（圧倒的に女性） |
| 年代 | **50代が最多**（50%以上）、次いで30〜40代 |
| 居住 | 1人暮らし（約1/3）、夫婦2人、実家暮らし |
| 間取り | ワンルーム〜4LDK、一軒家も |

### 共通する6つの問題パターン

1. **防災・備蓄強迫** — 不安から無限に備蓄（トイレットペーパー5年分等）
2. **整理収納の落とし穴** — アドバイザー資格を持つ人でも「整理」に留まり「削減」できない
3. **推し活・グッズ過多** — オタク文化でグッズ収集が正当化される
4. **子どもの物の過剰** — 「あると便利」で買い与えすぎ
5. **思い出の物への執着** — 服・グッズ・親の遺品が「心の重さ」として保管
6. **ストレス発散の買い物** — 心の不安定さが消費行動に表れる

### メンタルヘルスとの関連
- うつ病（みこさん47歳、29歳大阪）
- 不安症（ルルさん54歳 — 備蓄過多の背景）
- 介護ストレス（認知症のおばあちゃんの影響）
- ADHD傾向（塩沼さん38歳）

→ **しぶさんのコーチングは「物の問題」ではなく「心の問題」を扱っている**

## 4. コンテンツ戦略としての分析

### なぜこの動画フォーマットが10万〜68万回再生されるか

1. **冒頭の衝撃** — 「お邪魔します→やばい」で3秒以内に視聴者を掴む
2. **ドキュメンタリー性** — 2〜7日の密着で人間ドラマが生まれる
3. **変化の可視化** — ビフォーアフターの満足感
4. **共感性** — 50代女性（視聴者の主要層と一致）の悩みに寄り添う
5. **名言の繰り返し** — 「迷ったら手放す」等が視聴者の記憶に残る
6. **LINE導線** — 全動画の末尾で一貫してLINE登録を促進

### 編集上のミニマリズム
- BGMなし、効果音なし、テロップ最小限（りくとの編集哲学）
- カットのみで構成 — 6〜7年間このスタイルで再生数を維持
- 「何を残して何を削ぎ落とすか」= コンテンツ制作にもミニマリズム
